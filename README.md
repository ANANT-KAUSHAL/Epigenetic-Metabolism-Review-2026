This repository contains R scripts for simulated spatial transcriptomics analysis, corresponding to Figure 2 in the manuscript Epigenetic Regulation of Secondary Metabolism.

Contents
scripts/
Figure2_Spatial_Simulation.R:

Simulates normalized count data (TPM) for flavonoid biosynthesis genes.

Models differential chromatin states between glandular trichomes and mesophyll cells.

Generates hierarchical clustering heatmaps.

figures/
Figure2_Spatial_Heatmap.png: Output visualization generated by the script.

Dependencies
The analysis requires R (>= 4.0.0) and the following package:

R
install.packages("pheatmap")
Usage
To reproduce the figure:

Clone the repository.

Set the working directory to the repository root.

Run the script:

R
source("scripts/Figure2_Spatial_Simulation.R")
